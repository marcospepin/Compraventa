# PLANIFICACIÓN - MCR MOTORS

## 1. Metodoloxía Prevista

### 1.1. Modelo de Desenvolvemento

O proxecto MCR Motors seguirá un **modelo iterativo e incremental**, combinando elementos de metodoloxías áxiles adaptadas ao contexto dun proxecto individual de Formación en Centros de Traballo.

**Características principais:**

1. **Iterativo:**
   - Desenvolvemento por ciclos curtos
   - Revisión e mellora continua
   - Validación progresiva de funcionalidades

2. **Incremental:**
   - Entrega de prototipos funcionais en cada fase
   - Engadir funcionalidades de forma progresiva
   - Priorización de funcionalidades esenciais primeiro

3. **Control de versións:**
   - Uso de Git para rexistro de cambios
   - Commits frecuentes con mensaxes descriptivas
   - Etiquetado de versións (prototipos)
   - Repositorio en GitHub para respaldo

4. **Probas continuas:**
   - Validación de cada funcionalidade ao implementarse
   - Probas manuais en cada iteración
   - Corrección inmediata de erros detectados

### 1.2. Fases do Proxecto

O proxecto divídese en 5 fases principais:

1. **Estudo Preliminar:** Análise e definición do proxecto
2. **Análise:** Especificación detallada de requirimentos
3. **Deseño:** Arquitectura, base de datos e interfaces
4. **Codificación e Probas:** Implementación e validación
5. **Manuais:** Documentación técnica e de usuario
