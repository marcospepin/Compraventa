# ANTEPROXECTO - MCR MOTORS

## 1. Descrición do Proxecto

### 1.1. En que consiste o proxecto

MCR Motors é unha aplicación web de compraventa de vehículos de alta gama e deportivos que permite aos usuarios explorar, buscar e adquirir vehículos de segunda man de calidade premium. A plataforma ofrece un catálogo completo de vehículos con información detallada, imaxes, características técnicas e opcións de financiación personalizadas.

### 1.2. Propósito Principal

O propósito principal de MCR Motors é proporcionar unha plataforma dixital moderna e intuitiva que conecte compradores interesados en vehículos de alta gama cun concesionario especializado, facilitando o proceso de compraventa mediante:

- Visualización detallada de vehículos dispoñibles
- Sistema de xestión de favoritos
- Información sobre financiación
- Contacto directo co concesionario
- Panel de administración para xestión de inventario

## 2. Contextualización do Proxecto

### 2.1. A quen vai destinado

**Contexto social e sector empresarial:**
- Sector da automoción de alta gama e vehículos deportivos
- Mercado de vehículos de segunda man premium
- Ámbito xeográfico: Galicia (San Ciprián De Viñas, Ourense) con alcance nacional

**Tipo de persoas destinatarias:**
- **Compradores potenciais:** Persoas de 25-55 anos con poder adquisitivo medio-alto, interesadas en vehículos deportivos, de alta gama ou clásicos
- **Entusiastas do motor:** Coleccionistas e aficionados aos coches deportivos
- **Profesionais:** Persoas que buscan vehículos premium para uso profesional ou empresarial

**Organización:**
- MCR Motors é un concesionario familiar especializado en compraventa de vehículos de alta gama
- Presenza en plataformas como Wallapop, Coches.net e Instagram

**Produtos e servizos:**
- Compraventa de vehículos de alta gama e deportivos
- Asesoramento personalizado
- Opcións de financiación a medida
- Servizo postvenda

### 2.2. Necesidades que se pretenden cubrir

1. **Presenza dixital profesional:** Moitos concesionarios pequenos carecen dunha web propia profesional, dependendo únicamente de plataformas terceiras
2. **Centralización da información:** Os clientes deben buscar en múltiples plataformas para ver o inventario completo
3. **Transparencia e confianza:** Os compradores necesitan información completa e verificable dos vehículos
4. **Facilidade de contacto:** Proceso sinxelo para contactar co vendedor e consultar información
5. **Xestión de favoritos:** Posibilidade de gardar vehículos de interese para comparar
6. **Información sobre financiación:** Acceso claro ás opcións de financiamento

### 2.3. Análise de aplicacións existentes

**Competencia directa:**

1. **Coches.net**
   - Amplo alcance e visibilidade
   - Interface complexa con moita publicidade
   - Non personalizada para cada concesionario
   - **Limitación:** Falta de identidade de marca propia

2. **Wallapop**
   - Gran base de usuarios
   - Enfocada en transaccións C2C
   - **Limitación:** Pouca profesionalidade percibida, non especializada en automoción

3. **AutoScout24**
   - Plataforma internacional consolidada
   - **Limitación:** Comisións elevadas, pouca personalización

4. **Webs de concesionarios individuais**
   - Identidade de marca propia
   - Control total do contido
   - **Limitación:** Moitas están desactualizadas ou teñen mal deseño

**Conclusión:**
As plataformas terceiras ofrecen visibilidade pero non permiten crear unha marca propia nin controlar a experiencia do usuario. As webs propias de concesionarios pequenos adoitan estar desactualizadas. MCR Motors combina o mellor dos dous mundos: presenza propia profesional mantendo a presenza en plataformas terceiras.

### 2.4. Oportunidade de negocio e comercialización

**Oportunidade de negocio:**

1. **Modelo B2C (Business to Consumer):** Venda directa de vehículos a clientes finais a través da plataforma
2. **Modelo de servizo SaaS:** Potencial para adaptar a plataforma para outros concesionarios pequenos
3. **Marketing dixital:** A presenza web permite campañas de publicidade online (Google Ads, Facebook Ads)
4. **Fidelización:** Sistema de usuarios rexistrados permite crear base de datos de clientes potenciais

**Comercialización:**

1. **Fase inicial:** Uso interno para MCR Motors
2. **Fase de crecemento:** 
   - SEO para aparecer en buscas de "coches de alta gama Galicia"
   - Integración con redes sociais para ampliar alcance
   - Newsletter para clientes potenciais
3. **Fase de expansión:** 
   - Comercializar a plataforma como solución "white label" para outros concesionarios
   - Modelo de subscrición mensual para outros negocios similares

**Viabilidade:**
- **Investimento inicial baixo:** Non require hardware específico, úsase hosting compartido
- **Escalable:** Pódese comezar con poucos vehículos e crecer gradualmente
- **Retorno de investimento:** Cada venda xustifica o custo de mantemento da web

## 3. Análise Preliminar do Proxecto

### 3.1. Obxectivos do proxecto

**Obxectivos principais:**
1. Crear unha presenza dixital profesional para MCR Motors
2. Centralizar toda a información dos vehículos en venda
3. Facilitar o contacto entre compradores e o concesionario
4. Proporcionar información clara sobre financiación
5. Mellorar a imaxe de marca e profesionalidade percibida

**Obxectivos secundarios:**
1. Reducir o tempo de resposta a consultas mediante formularios web
2. Proporcionar información 24/7 aos potenciais compradores
3. Crear unha base de datos de clientes potenciais
4. Mostrar vehículos vendidos como portfolio
5. Facilitar a xestión do inventario mediante panel de administración

### 3.2. Alcance do proxecto

**Funcionalidades incluídas:**
- Web pública con catálogo de vehículos
- Sistema de busca e filtrado
- Páxinas informativas (financiación, contacto)
- Sistema de favoritos para usuarios rexistrados
- Formularios de contacto
- Panel de administración para xestión de vehículos
- Galería de vehículos vendidos
- Sistema de autenticación de usuarios
- Responsive design para dispositivos móbiles

**Fora do alcance (fase inicial):**
- Pasarela de pago online
- Sistema de reserva de citas
- Chat en tempo real
- App móvil nativa
- Sistema de valoración de vehículos
- Multiidioma (posible mellora futura)

### 3.3. Requisitos principais

**Requisitos funcionais:**
- Xestión completa de vehículos (CRUD)
- Sistema de usuarios con roles (admin/cliente)
- Carga e xestión de imaxes
- Formularios de contacto funcionais
- Sistema de favoritos persistente
- Filtros de busca por marca, modelo, prezo, etc.

**Requisitos non funcionais:**
- Rendemento: Tempo de carga inferior a 3 segundos
- Usabilidade: Interface intuitiva e responsive
- Seguridade: Protección de datos segundo LOPDGDD
- Mantemento: Código limpo e documentado
- Compatibilidade: Funcionar en todos os navegadores principais
