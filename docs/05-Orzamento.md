## 4. Ingresos Esperados

MCR Motors é un concesionario de vehículos de alta gama. A web non xera ingresos directos (non hai pago online), pero actúa como ferramenta de **xeración de leads e mellora de vendas**.

### 4.1. Modelo de Xeración de Valor

A web contribúe aos ingresos de MCR Motors de forma indirecta:

1. **Aumento de visibilidade:** Máis visitas = máis leads
2. **Profesionalización da marca:** Maior confianza = maior taxa de conversión
3. **Disponibilidade 24/7:** Información sempre accesible
4. **Xeración de base de datos:** Usuarios rexistrados = potenciais compradores

### 4.2. Estimación de Impacto en Vendas

#### 4.2.1. Datos do Negocio

| Métrica | Valor |
|---------|-------|
| Prezo medio de vehículo vendido | 25.000€ |
| Marxe bruto medio por venda | 3.000€ |
| Vendas anuais actuales (sen web) | 20 vehículos |
| Ingresos brutos anuais actuales | 60.000€ |

#### 4.2.2. Incremento Esperado con Web

**Hipótese conservadora:**
- Incremento de vendas: 15% (3 vehículos adicionais)
- Mellora de marxe por maior confianza: 5%

| Concepto | Sen Web | Con Web | Diferenza |
|----------|---------|---------|-----------|
| Vehículos vendidos/ano | 20 | 23 | +3 |
| Marxe medio por vehículo | 3.000€ | 3.150€ | +150€ |
| **Ingresos brutos totais** | **60.000€** | **72.450€** | **+12.450€** |
